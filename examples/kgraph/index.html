demo-kgraph.wasm (kgraph-* host imports)

demo-kgraph.wasm is the byte-for-byte output of kotoba wasm emit src/demo_kgraph.kotoba --policy src/demo_kgraph_policy.edn in kotoba-lang/kotoba (219 bytes; declares the kgraph_assert/ kgraph_query host imports). This example wires those imports to kgraph.js -- a browser-side port of kotoba-lang/kotoba's in-memory EAVT datom store -- via KotobaWasmElement.define's createImports option; the module asserts [1 :name "Aoi"] then queries {:find [?v] :where [[1 :name ?v]]}, entirely inside this page.
